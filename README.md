# installation 
ouvrez votre terminal et exceuter les commandes suivantes:
> git clone https://github.com/Eliekationan/API-BLOCKCHAIN
> cd API-BLOCKCHAIN-main
> npm install
pour lancer l'application fait "npm run dev"
